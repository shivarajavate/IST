Deployment Instructions

1. Install node.js
2. Install MongoDB
3. Using npm, install bower globally: npm install -g bower
4. Using npm, install grunt-cli globally: npm install -g grunt-cli
5. run: npm install
6. run: bower install
7. run: grunt dev
8. run: node index.js
9. run: node mongo.script.js

NOTE: To deploy
run: grunt deploy
